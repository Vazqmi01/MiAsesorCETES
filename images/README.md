# Carpeta de Imágenes

Esta carpeta contiene las imágenes utilizadas en la aplicación Mi Asesor CETES.

## Imágenes Utilizadas

La aplicación busca las siguientes imágenes (en orden de prioridad):

1. **hero.jpg** o **hero.png** - Imagen principal del banner/header
2. **investment.jpg** o **investment.png** - Imagen relacionada con inversiones
3. **growth.jpg** o **growth.png** - Imagen relacionada con crecimiento
4. **calculator.jpg** o **calculator.png** - Imagen relacionada con cálculos

## Formato Recomendado

- **Formato**: JPG o PNG
- **Resolución recomendada**:
  - Hero: 1200x400px
  - Otras imágenes: 800x600px
- **Tamaño de archivo**: Menor a 1MB por imagen

## Cómo Agregar Imágenes

1. Descarga o crea las imágenes relacionadas con finanzas, inversiones o CETES
2. Guarda las imágenes en esta carpeta (`images/`)
3. Nombra las imágenes según la lista anterior (hero.jpg, investment.jpg, etc.)
4. La aplicación cargará automáticamente las imágenes locales si existen
5. Si no existen imágenes locales, se usarán imágenes desde URLs públicas

## Fuentes de Imágenes

Puedes encontrar imágenes gratuitas en:
- [Unsplash](https://unsplash.com) - Búsqueda: "finance", "investment", "money"
- [Pexels](https://www.pexels.com) - Búsqueda: "business", "finance", "investment"
- [Pixabay](https://pixabay.com) - Búsqueda: "finance", "money", "investment"

## Notas

- Las imágenes locales tienen prioridad sobre las imágenes desde URLs
- Si no hay imágenes locales, la aplicación usará imágenes desde Unsplash
- Asegúrate de que las imágenes sean relevantes al tema de inversiones y CETES
